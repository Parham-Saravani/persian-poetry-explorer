# ✨ Avin | آوین

Avin is a modern Persian poetry web application built with Vanilla JavaScript and powered by the Ganjoor API.

The project provides an elegant experience for exploring Persian poets, reading poetry, and receiving Hafez fortunes through a responsive and modern user interface.

## ✨ Features

### 🏠 Home Page

- Hero slider
- Random Persian verse
- Responsive layout
- Dark and Light mode

### 🌙 Theme Switching

- Dark Mode
- Light Mode
- Theme persistence using Local Storage

### 👤 Poets Page

- Browse Persian poets
- Century-based filtering
- Responsive poet cards
- Fast navigation to poet profiles

### 📖 Poet Profile Page

- Poet biography
- Poet image
- Birth information
- Death information
- Birthplace information
- Poet works and collections
- Display poems and categories
- Skeleton loading states

### 🪶 Hafez Fortune

- Random Hafez fortune
- Fortune interpretation
- Elegant fortune reading experience

### 📱 Responsive Design

- Mobile friendly
- Tablet friendly
- Desktop optimized

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Parham-Saravani/avin.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## 🎯 Project Goals

Avin aims to make Persian poetry more accessible through a modern web experience and introduce users to the works of great Persian poets.

Supported poets include:

- Hafez
- Saadi
- Rumi
- Ferdowsi
- Khayyam
- And many more
